# Portfolio Assets

## Social Preview Image

Create a `social-preview.png` file (1200x630px) for social media sharing.

### Design Specifications

- **Dimensions**: 1200 x 630 pixels
- **Format**: PNG or JPG
- **File size**: Under 1MB for optimal loading

### Recommended Content

Include the following in your social preview:

1. **Your Name**: "Ruslan Maqueira"
2. **Title/Tagline**: "Full-Stack Software Engineer"
3. **Key Technologies**: Display 4-6 main tech stack items
   - .NET / C#
   - Angular / TypeScript
   - Azure OpenAI
   - SQL Server
   - Node.js
   - Docker

4. **Visual Elements**:
   - Professional gradient background (use your site's colors)
   - Clean, modern typography
   - Optional: Subtle code snippet or tech icons

### Design Tools

**Free Options**:
- **Canva**: https://canva.com
  - Search for "LinkedIn Banner" or "Twitter Header" templates
  - Resize to 1200x630px
  - Use template ID: Social Media Preview

- **Figma**: https://figma.com
  - Create new frame: 1200x630px
  - Export as PNG
